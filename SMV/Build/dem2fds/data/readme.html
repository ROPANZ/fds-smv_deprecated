Creating a Terrain Input File for Use with FDS

Preliminaries

Install or verify that the following is installed:
  • Imagemagick
  • latest installed or test smokeview

Create FDS Input File

  • Go to the USGS National Map website. You should see a map of the United States on the right and a set of data options on the left.
    1. Obtaining Elevation data
      1. Select the Elevation Products (3DEP) checkbox.
      2. Select 1/3 arc-second DEM checkbox and GridFloat File format radio button
      3. Click on the map and zoom in on a region of interest.
      4. Click on the Find Products button. You should see one or more entries listed with headings beginning with USGS NED 1/3 arc-second ...... 1 x 1 degree . Make sure that GridFloat is listed in heading (if not, click on Return to Search and verify you selected GridFloat option)
      5. Click on the Footprint link to verify where various files are located.
      6. Click on Download link to download files of interest.
      7. Unzip these files (using a program like winzip). Copy the .flt and .hdr files to a directory.
    2. Obtaining Image data
      1. Click on the Return to Search button and uncheck the Elevation Products (3DEP) checkbox
      2. Select the Imagery - 1 meter (NAIP) checkbox
      3. Click on the map and zoom in on a region of interest (same region selected for elevtaion files)
      4. Click on the Find Products button. You should see one or more entries listed with headings beginning with m_ and ending with .jp2 .
      5. Click on the Footprint link to verify where various files are located.
      6. Click on Download link to download files of interest.
      7. Copy the downloaded files to the same directory you copied the .flt and .hdr elevation files.
      8. The .jp2 files you just downloaded are in jpeg 2000 format. They need to be converted to jpeg format . To do this, run the following command (on a PC) for each jp2 file: 
          magick imagefilename.jp2 imagefilename.jpg
        Note, this Imagemagick command is named convert on a Linux or Mac.
    3. Creating an FDS input file
      1. create a file named casename.in containing the following two lines
        LONGLATMINMAX
  longmin  longmax nlongs latmin latmax nlats
        where longmin longmax are the minimum and maximum longitude of the region you are modeling, latmin latmax are the miniumum and maximum latitude and nlong and nlat are the number of longitudes and latitudes.
      2. Run the command:
         dem2fds -d elevation_directory -o casename.in
        where elevation_directory is the path for the directory containing the elevation and image files you downloaded.
      3. An FDS input file has been created. To view the case, run the commands: 
        fds casename.fds
smokeview casename
      Note, the FDS input file, casename.fds is only a starting point. You will need to edit it to model your scenario as you wish.
